# Kapp Documentation

Kapp is a multi-tenant ERP and business management suite. This site
collects the operator, developer, and administrator documentation for
the platform.

If you are landing here for the first time, start with one of:

- **Developer**: [Developer guide](./DEVELOPER_GUIDE.md), then
  [KType authoring guide](./KTYPE_AUTHORING_GUIDE.md).
- **Operator / SRE**: [Operator guide](./OPERATOR_GUIDE.md), then
  [Capacity planning](./CAPACITY_PLANNING.md) and
  [Scaling runbook](./SCALING_RUNBOOK.md).
- **On-call**: [Incident response](./INCIDENT_RESPONSE.md) and
  [On-call playbook](./ONCALL_PLAYBOOK.md).
- **Administrator**: [Admin guide](./ADMIN_GUIDE.md).
- **Compliance / Security**: [Security review](./SECURITY_REVIEW.md),
  [Security hardening](./SECURITY_HARDENING.md),
  [Compliance](./COMPLIANCE.md).
- **Architecture**: see the [ADR index](./adr/README.md). The
  high-level architecture and phase plan live at the repo root —
  [ARCHITECTURE.md](https://github.com/kennguy3n/kapp-fab/blob/main/ARCHITECTURE.md)
  and
  [PHASES.md](https://github.com/kennguy3n/kapp-fab/blob/main/PHASES.md).

The documentation site is built with MkDocs Material; see
[Documentation site](./DOCS_SITE.md) for how to build and deploy
this site locally and in CI.
